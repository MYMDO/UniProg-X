# UniProg-X: Universal Hardware Programmer

UniProg-X is a next-generation, hyper-scalable, and automated hardware programmer designed for the modern era.

## Features
- **Universal Support**: I2C EEPROM, SPI Flash, NAND Flash, and more.
- **Web-First Interface**: No drivers needed. Works directly in Chrome/Edge via WebSerial.
- **High Performance**: Powered by RP2040 PIO for maximum throughput.
- **Modern UI**: Beautiful, responsive, and intuitive interface.

## Architecture
- **Firmware**: C++ (Arduino/Pico SDK) on RP2040.
- **Client**: React + TypeScript + Vite.
